# Midterm Seatwork #2 – Smart Wallet System

## Program Task

Create a Java program that simulates a digital wallet system with transaction tracking.

## Program Name

Create a Java program named:
**SmartWalletSystemLastName**  
Example: **SmartWalletSystemBedis**

## Initial Balance

- 5,000 pesos

## Menu Display

```bash
===== SMART WALLET MENU =====

1. View Balance
2. Cash In
3. Send Money
4. Exit
```

## Additional Requirement

&nbsp;- Count and display the number of successful transactions only

## Program Rules

Your program must use:

- while loop (continuous process)
- switch-case (menu selection)
- if-else (validation rules)
- Transaction counter
- Scanner class

## Important Conditions

### Send Money Rules

- Minimum amount: 100 pesos
- Maximum per transaction: 1,500 pesos

### Do NOT allow transaction if:

- Amount is less than 100
- Amount exceeds 1,500
- Amount exceeds current balance

## Handwritten Requirement

The Java program must be written neatly by hand on paper.

### Follow these rules:

- Use clear and readable handwriting
- Observe proper indentation and spacing

## Sample Execution (at least 3 scenarios)

```bash
Transaction 1:
Choice: 1
Output: Balance = 5000

Transaction 2:
Choice: 2
Cash In: 2000
Output: New Balance = 7000

Transaction 3:
Choice: 3
Send Money: 2000
Output: Exceeds maximum limit

Transaction Count: 2
```

## Short Explanation (3–5 sentences)

Explain:

- How the while loop keeps the program running
- How switch-case controls menu selection
- How if-else validates transactions
