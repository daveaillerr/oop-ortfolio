# Midterm Activity #3 – Basic ATM System

## Objective

Create a Java program that accepts student information and evaluates the student’s grade using decision structures.

## Program Requirements

1. Create a Java program named `BasicATMSystemLastName`.
2. Use the `Scanner` class to accept user input.
3. Assume the initial balance is **5,000 pesos**.

## Display the Menu

```bash
===== ATM MENU =====
1. Check Balance
2. Deposit
3. Withdraw
4. Exit
```

## Program Rules

- Use a **switch statement** to process the selected option.
- Use **if–else statements** to validate withdrawal.
- Maximum withdrawal per transaction: **2,000 pesos**.
- The program must repeat using a **do-while loop** until the user selects **Exit**.

## Submission Format

- Submit the Java source code file (`.java`).
- The filename must follow this format:  
  `Section_LastName_FirstName_ActivityName.java`
- Include screenshots of the program output showing the following:
  - User input
  - Program processing
  - Final output/result
- The screenshot must clearly display the entire console output.
