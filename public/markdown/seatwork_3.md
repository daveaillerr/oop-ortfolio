# Midterm Seatwork #3: Student Age Analyzer

## Instructions

Create a Java program that evaluates a student’s final grade using:

- Scanner (for user input)
- User-defined methods
- Control structures (if-else / switch)

## Program Requirements

1.  Create a Java class named:  
    StudentAgeAnalyzerLastName  
    (Example: StudentAgeAnalyzerBedis)

2.  Ask the user to input:  
    o Student Name  
    o Age

3.  Use a user-defined method to:  
    o Determine the age category

4.  Use if-else conditions:

| Age Range | Category       |
| --------- | -------------- |
| 0-12      | Child          |
| 13-19     | Teenager       |
| 20-59     | Adult          |
| 60 above  | Senior Citizen |

## Handwritten Requirement

The Java program must be written neatly by hand on paper

Follow these rules:

- Use clear and readable handwriting
- Observe proper indentation and spacing
- Write all parts of the program:
  - import statement
  - class declaration
  - user-defined methods
  - main method

Avoid:

- Erasures and messy corrections
- Illegible handwriting

**Note:** Unreadable code will not be checked and may receive a score of 0
